== Description ==

Official Plugin to seamlessly integrate with Graziashop.com, the luxury online global shopping destination.

Made in collaboration with Republik Media Ltd (http://republik-media.com).

== Installation ==

1. Upload 'graziashop-integration' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate through your products details pages to see the custom fields and decide whether or not to make them sellable on Graziashop.com
4. Click on the new menu item "Generate CSV" to generate the CSV file which will be used to integrate with Graziashop
